<h1>Welcome to DeepUI.dev</h1>

Almost all components are built using [Frosted-UI](https://www.npmjs.com/package/frosted-ui), a React component library designed for building modern web applications with a focus on accessibility and ease of use.
This documentation site serves as a showcase for the components available in the DeepUI library, providing examples and usage guidelines to help developers integrate these components into their projects effectively.
Explore the various components, from basic UI elements to more complex structures, and discover how DeepUI can enhance your application's user interface.
